Heart Disease Risk Analysis

Investigated risk factors associated with heart disease using a large clinical dataset. Applied logistic regression and machine learning models to identify key predictors such as age, cholesterol, and blood pressure. Results were visualized in an interactive report to support evidence-based understanding of cardiovascular health patterns in the general population.

Reproducible Data Analysis Pipeline

Built a fully reproducible data analysis pipeline using R Markdown, covering data cleaning, exploratory analysis, and statistical modeling. The project emphasized best practices in computational reproducibility, including version control and automated report generation — skills essential for transparent and replicable public health research.
